
## 一、剧本解析与场次拆解

**操作步骤：**

1. 使用提示词 `【提取场次角色场景】` 将完整剧本输入大语言模型（推荐：Gemini(只能解析一段或者一集，上下文推理不出整个剧本)、ChatGPT（最强推理）、Kimi（比较细心，内容很丰富）、DeepSeek（可以推理整个剧本，但是分镜太常规））。  
   > ⚠️ 避免使用豆包等推理能力较弱的模型。

2. 筛选出需要推理生成的场次，明确每场戏的**角色**、**场景**、**视觉风格**与**道具**。
3. 按需求去分析，根据不同的模型去选择最好的人物设计场景设计，可以多模型互用；

## 三、场景生成规范

- **模型优先级**：  
  1️⃣ GPT-image-2（首选）  
  2️⃣ nano banana（次选）  
  🚫 其他模型一律禁止，如有特殊需求须经主管书面确认。

- **技术参数**：
  - 角度：**俯视**
  - 比例：**21:9**
  - 光影：室内外均需**明确光源位置**，所有场景必须经过**压光处理**，保持 **“只暗不亮”** 原则。

> ✅ 室内：清晰的光影分区（如窗户光、台灯光斑）  
> ✅ 室外：明确日光或月光方向，影子浓度与光源匹配

## 五、视频生成（核心流程）

### 5.1 主模型

- **统一使用**：`seedance 2.0`  
- **次选方案**：其他视频模型须经**主管确认**后方可使用。

### 5.2 生成前自检清单（⚠️必须逐条过脑）

#### A. 段落衔接
- 本次生成从哪一段开始，哪一段结束？
- 下一片段从哪个动作/台词/镜头接上？
- 前后两段的**镜头运动、视线方向、情绪节奏**是否流畅？

#### B. 基础镜头参数
- ✅ 镜头是否运动？（除必要外，所有镜头必须运动，严禁固定镜头）
- ✅ 近景 / 中景是否使用**大光圈、浅景深**？
- ✅ 是否有**逆光 / 发丝光 / 轮廓光 / 丁达尔效应 / 光斑 / 伦勃朗光**？（除非必要，所有镜头必须有光影层次）
- ✅ 是否明确标注了**光源位置**（如“右侧窗外暖光”“左侧逆光冷调”）？

#### C. 角色与空间
- 人物站位是否正确与镜头运动是否搭配？
- 后景是什么（不空、不糊弄）？
- 氛围是什么（压抑、温馨、紧张、梦幻）？
- 环境状态（风、尘埃、烟雾、水汽、落叶等）？

#### D. 剪辑与时长
- 镜头衔接是否考虑**动作点**（尾帧预留剪辑余地）？
- 片段时长是否足够覆盖台词？
- 台词是否配置了**情绪标签**（如「咬牙切齿」「平静淡漠」「气声低语」）与语速提示？
- 音效是否已预设（风声、脚步声、衣料摩擦等）？

#### E. 高级镜头技法（按需选择）
- 是否有**前景**？（除全景外，尽量安排前景物体，增强空间感）
- 是否需要**升格**（慢动作）？
- 是否需要**荷兰角**（倾斜构图）？
- 是否需要**广角镜头**（增强透视变形）？
- 是否明确了**镜头运动方向**（推/拉/摇/移/跟/升降）？
- 人物与镜头运动是否**匹配**（如人物左移，镜头是否跟摇）？
